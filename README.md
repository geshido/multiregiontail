Multi-region tail
===

A tool for tailing CloudWatch logs of different regions at once

Install
---

`go get github.com/geshido/multiregiontail`

Run
---

```
Usage of multiregiontail:
  -group string
    	log group name
  -profile string
    	regs, comma separated (default "default")
  -regs string
    	regs, comma separated
```
